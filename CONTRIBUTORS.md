## Contributing

We welcome contributions from the community! To add your name to this list:

1. Make a meaningful contribution to the project
2. Submit a pull request
3. Your contribution will be reviewed
4. Upon approval, you will be added to this file

See [CONTRIBUTING.md](CONTRIBUTING.md) for contribution guidelines.

---

## Recognition Categories

### Code Contributors
Individuals who have contributed code, documentation, or infrastructure improvements.

### Issue Reporters
Community members who have reported bugs, suggested features, or provided valuable feedback.

### Security Researchers
Individuals who have responsibly disclosed security vulnerabilities.

### Documentation Contributors
Contributors who have improved documentation, guides, or examples.

---

## How to be Listed

To be added to this file, you must:
- Make at least one merged pull request, OR
- Report a valid bug that leads to a fix, OR
- Responsibly disclose a security vulnerability, OR
- Contribute significant documentation improvements

**Format:**
```markdown
**Your Name** - Brief description
- GitHub: @username
- Contribution: Description of contribution
```
